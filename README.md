# node-es6-skeleton

> Mode.js skeleton with ES6 for Server Projects with Mongoose.

## Install

```
$ npm install
$ npm install -g nodemon // dev
$ npm install -g pm2 // start
```

## Run Development Server

```
$ npm run dev
```
